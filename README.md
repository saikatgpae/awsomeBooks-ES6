# Awesome books

> "Awesome books" is a simple website that displays a list of books and allows you to add and remove books from that list.

## Built With

- Major languages: HTML, CSS, JavaScript.
- Frameworks: Bootstarp, luxon.
- Technologies used: Linters, Bootstrap.


[Live Demo Link](https://saikatgpae.github.io/awsomeBooks-ES6/)




## Getting Started


To get a copy in your local mechine:

Clone this repository or download the Zip folder:
git clone https://github.com/saikatgpae/awsomeBooks-ES6.git
